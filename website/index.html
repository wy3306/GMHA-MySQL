03 · Architecture

控制面集中，
执行面贴近实例。

Manager 负责任务编排、状态管理与 Web 控制台；Agent 部署在受管主机，贴近 MySQL 执行采集与变更。两者通过多种通道协同，在可达性与实时性之间取得平衡。

  • 01
    gRPC 双向流承载低延迟心跳与动态采集配置
  • 02
    WebSocket 任务通道执行过程持续回传，状态实时可见
  • 03
    SSH 兜底恢复用于主机纳管、Agent 部署与离线恢复
